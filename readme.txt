== Description ==

A plugin that places a simple chatbox widget in your sidebar.

It uses pure php, no scripts used.

It is recommended that any old version is uninstalled through the WordPress plugins page before installing a new version.

== Installation ==

1. Upload the 'chatbox' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In the apperearance manu, wrag the widget into your sidebar, change settings as you like and click save
